# 🎵 موزیک استریم - سایت پخش موزیک آنلاین

یک سایت کامل پخش موزیک آنلاین شبیه Spotify که برای اجرا روی cPanel طراحی شده است.

## ✨ ویژگی‌های اصلی

- پخش‌کننده پیشرفته با کنترل‌های کامل
- پشتیبانی از فرمت‌های MP3, WAV, FLAC
- پخش تصادفی و تکرار
- کنترل صدا و پیشرفت پخش
- صف پخش قابل مدیریت

### 👤 مدیریت کاربران
- سیستم ثبت نام و ورود
- نقش‌های مختلف (ادمین، کاربر عادی، VIP)
- پروفایل کاربری قابل ویرایش
- تاریخچه پخش

### 🎵 مدیریت موزیک
- آپلود و مدیریت آهنگ‌ها
- دسته‌بندی بر اساس ژانر
- اطلاعات کامل آهنگ (هنرمند، آلبوم، مدت زمان)
- سیستم لایک و نظرات

### 📚 پلی‌لیست‌ها
- ایجاد پلی‌لیست شخصی
- اشتراک‌گذاری پلی‌لیست‌ها
- پلی‌لیست‌های پیشنهادی
- مدیریت آسان آهنگ‌ها در پلی‌لیست

### 🔍 جستجو و فیلتر
- جستجوی پیشرفته در آهنگ‌ها
- فیلتر بر اساس ژانر، هنرمند، آلبوم
- پیشنهادات هوشمند
- جستجوی صوتی

### 🎛️ پنل مدیریت جامع
- داشبورد با آمار کامل
- مدیریت کاربران
- مدیریت موزیک و آپلود فایل
- مدیریت پلی‌لیست‌ها
- مدیریت محتوا (صفحات، بنرها، اعلانات)

## 🚀 نصب و راه‌اندازی

### مرحله 1: آپلود فایل‌ها

1. تمام فایل‌های پروژه را دانلود کنید
2. فایل‌ها را در دایرکتوری اصلی وب‌سایت (public_html) آپلود کنید
3. مجوزهای فایل‌ها را بررسی کنید

```bash
# مجوزهای پیشنهادی
chmod 755 folders/
chmod 644 files
chmod 777 uploads/ (و زیرپوشه‌هایش)
```

### مرحله 2: ایجاد پایگاه داده

1. از cPanel وارد phpMyAdmin شوید
2. یک پایگاه داده جدید ایجاد کنید (مثلاً: `music_stream_db`)
3. محتویات فایل `database.sql` را در پایگاه داده اجرا کنید

### مرحله 3: تنظیم پایگاه داده

فایل `config/database.php` را ویرایش کنید:

```php
private $host = 'localhost';
private $db_name = 'music_stream_db';
private $username = 'your_db_username';
private $password = 'your_db_password';
```

### مرحله 4: ایجاد پوشه‌های آپلود

```bash
mkdir uploads
mkdir uploads/music
mkdir uploads/images
mkdir uploads/covers
mkdir uploads/avatars
chmod 777 uploads -R
```

### مرحله 5: تنظیمات امنیتی

1. کلید JWT را در `config/database.php` تغییر دهید:
```php
define('JWT_SECRET', 'your_unique_secret_key_here');
```

2. اطلاعات سایت را به‌روزرسانی کنید:
```php
define('SITE_URL', 'https://yourdomain.com');
define('ADMIN_EMAIL', 'admin@yourdomain.com');
```

### مرحله 6: دسترسی به پنل مدیریت

- آدرس پنل مدیریت: `https://yourdomain.com/admin.html`
- نام کاربری پیش‌فرض: `admin`
- رمز عبور پیش‌فرض: `admin123`

⚠️ **مهم**: حتماً رمز عبور ادمین را در اولین ورود تغییر دهید!

## 🔧 تنظیمات پیشرفته

### تنظیم حداکثر حجم آپلود

فایل `.htaccess` ایجاد کنید:

```apache
php_value upload_max_filesize 50M
php_value post_max_size 50M
php_value max_execution_time 300
php_value max_input_time 300
```

### فعال‌سازی HTTPS

```apache
RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]
```

### بهینه‌سازی سرعت

1. فعال‌سازی Gzip compression
2. استفاده از CDN برای فایل‌های استاتیک
3. بهینه‌سازی تصاویر
4. کش کردن فایل‌های CSS/JS

## 🔒 امنیت

- تمام ورودی‌ها sanitize می‌شوند

## 🐛 عیب‌یابی

### مشکلات رایج:

1. **آپلود فایل کار نمی‌کند**
   - مجوزهای پوشه uploads را بررسی کنید
   - تنظیمات PHP upload را چک کنید

2. **اتصال به پایگاه داده برقرار نمی‌شود**
   - اطلاعات اتصال در database.php را بررسی کنید
   - دسترسی‌های پایگاه داده را چک کنید

3. **صفحه admin لود نمی‌شود**
   - مسیر فایل‌های CSS/JS را بررسی کنید
   - Console browser را چک کنید
